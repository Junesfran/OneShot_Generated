## Pa linux
### El python primero
Primero por ser limpito y ordenado, el venv:
```bash
python3 -m venv .venv
```
activarlo:
```bash
source .venv/bin/activate
```
instalar la paketeria:
```bash
pip install -r requirements.txt
```
y lanzar la API sin miedo al exito:
```bash
python3 run.py
```
